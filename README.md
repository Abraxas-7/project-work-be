## 🛠️ Installazione

1. Installa le dipendenze:

   ```bash
   npm install
   ```

2. Configura il file delle variabili d’ambiente:  
   Crea un file `.env` e aggiungi:

   ```env
   PORT = [la tua porta]

   DB_HOST = localhost
   DB_USER= root
   DB_PASSWORD= [la tua password]
   DATABASE= [il tuo database]
   ```

3. Avvia il server:
   ```bash
   npm run dev
   ```
